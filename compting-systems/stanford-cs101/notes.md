## Bits, Bytes & Binary
Everything in the computer is stored in binary which consists of only 2 values / states which are 0's and 1's or on and off. These binaries can be combined to form different numbers which will have the bases of 2. 1 byte consists of 8 bits, 8 different combination of 0's and 1's, which can lead up to the values from 0 till 255. <br>

## Storing Bits and Bytes in other media
- Solid State Drive (SSD) - uses electronic switches
- Hard Disk Drive (HDD) - magnetically polarizes sections of disk
- CD, DVD, Blu-ray - pits and lands on disks representing 0's and 1's
